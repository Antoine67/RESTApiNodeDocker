# REST Api réalisé avec NodeJS/Typescript et conteneurisation via Docker/HAProxy

Installation :

`docker build -t node_eat .`

`docker-compose up` 

API accessible sur `http://{ip_machine}:8080`

Documentation API accessible sur `http://{ip_machine}:8080/docs`

Versions utilisées :
`node v14.17.0`
`npm v6.14.13`
`docker v20.10.2`
